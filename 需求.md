
# 社交媒体平台需求文档

## 项目概述
一个简单的社交媒体平台，用户可以注册、登录，发布动态，查看关注用户的时间线，互动（点赞、评论），并能接收到通知。项目前端采用 **React** + **TailwindCSS**，后端采用 **Node.js + Express** 或 **Firebase**，推荐使用 **Next.js** 提升开发效率。

---

## 1. 需求分析

### 1.1 前端需求
#### 用户模块
1. **注册/登录**
   - 用户通过邮箱和用户名注册。
   - 登录时使用邮箱/用户名+密码。
   - 登录成功后跳转到时间线页。

2. **个人主页**
   - 展示用户的头像、昵称、简介、已发布动态。
   - 支持关注和取消关注其他用户。
   - 提供修改个人资料页面。

3. **用户设置**
   - 支持修改头像、昵称、密码、简介。

#### 动态模块
1. **发布动态**
   - 支持文本发布。
   - 支持多张图片或视频上传。
   - 上传媒体后提供预览功能。

2. **动态互动**
   - 点赞、取消点赞动态。
   - 对动态发表评论或回复他人评论。

3. **时间线**
   - 主页面展示用户关注的人的动态。
   - 按时间倒序排列。
   - 支持无限滚动加载。

#### 通知模块
1. 显示新评论、点赞和关注的通知。
2. 支持清除已读通知。

#### 搜索模块
1. 搜索用户：输入用户名关键词，展示匹配用户。
2. 搜索动态：根据关键词或标签展示匹配动态。

---

### 1.2 后端需求
#### 用户模块
1. 用户注册/登录 API。
2. 获取/更新用户资料 API。
3. 关注/取消关注用户 API。

#### 动态模块
1. 发布动态 API，支持文本和多媒体上传。
2. 点赞/取消点赞 API。
3. 评论/回复 API。
4. 获取时间线 API，支持分页。

#### 通知模块
1. 获取未读通知 API。
2. 更新通知状态 API（标记为已读）。

#### 搜索模块
1. 用户搜索 API。
2. 动态搜索 API。

## 3. 原型图
### 首页 - 时间线
- 显示关注用户的动态。
- 动态卡片包括：用户头像、用户名、发布时间、文本内容、图片/视频、点赞数、评论数。
- 无限滚动加载。

### 发布动态页
- 文本输入框。
- 图片/视频上传区域。
- 发布按钮。

### 个人主页
- 用户头像、昵称、简介。
- 用户的所有动态（按时间倒序）。
- 关注按钮。

### 登录/注册页面
- 邮箱、用户名、密码输入框。
- 登录/注册切换选项。

### 通知页面
- 列表形式展示所有通知。
- 未读通知高亮显示。

### 搜索页面
- 搜索框。
- 展示用户或动态的搜索结果列表。

## 4. 技术选型
| 模块          | 技术                          | 说明                      |
|---------------|-------------------------------|---------------------------|
| 前端框架      | React + TailwindCSS           | 快速构建响应式页面         |
| SSR/SEO 优化  | Next.js                       | 提高性能和 SEO 效果        |
| 状态管理      | Context API / Redux Toolkit   | 小型项目推荐 Context API   |
| 后端框架      | Node.js + Express / Firebase  | 自行构建或使用云后端       |
| 数据库        | MongoDB / Firebase Firestore  | 非关系型数据库             |
| 实时功能      | WebSocket (Socket.io)         | 实现通知和私信             |
| 部署平台      | Vercel / Heroku               | 部署前端和后端             |
| 媒体存储      | Cloudinary / Firebase Storage | 处理图片和视频优化         |

## 5. 开发计划

### 阶段 1：基础功能
1. 项目初始化（Next.js + TailwindCSS）。
2. 用户注册和登录功能实现。
3. 时间线页面基础布局和假数据展示。

### 阶段 2：动态模块
1. 实现动态发布功能。
2. 支持图片上传和预览。
3. 点赞和评论功能。

### 阶段 3：用户模块
1. 个人主页布局和动态展示。
2. 关注/取消关注功能。

### 阶段 4：通知和搜索模块
1. 基本通知功能实现。
2. 搜索用户和动态功能。

### 阶段 5：优化和部署
1. 响应式设计优化。
2. 国际化支持（多语言）。
3. 部署项目到 Vercel 和云存储。
